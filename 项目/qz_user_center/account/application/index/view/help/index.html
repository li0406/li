注册齐装网帐号时遇到问题了？
  • 注册齐装网帐号时遇到问题了？
    您可以在PC端首页点击【登录】-【立即注册】根据提示注册齐装网帐号。
  • 新办理的手机已被注册，使用手机登录时提示因安全原因被封禁，怎么办？
    您可以在齐装网前台“在线客服咨询”寻求帮助。
  • 注册时，提示昵称已被使用，怎么办？
    亲爱的齐装网用户，齐装帐号昵称是唯一的，不可重复设置，且设置成功后无法修改，建议您编辑其他名称设置昵称。
  • 手机号已注册齐装网帐号，如何重新注册帐号？
    亲爱的齐装网用户，一个手机号注册后，可在齐装系产品通用，因此无需再重新注册账号。
登录帐号时遇到问题了？
  • 为何提示帐号不存在？
    齐装网帐号统一后，部分未绑定手机的帐号无法直接登录，您可以再次快速注册通用帐号。
  • 使用第三方帐号登录，如何查看昵称和头像？
    点击头像可跳转至基本信息页面，查看当前登录帐号头像和昵称，建议您编辑其他名称设置昵称。
  • 使用第三方帐号登录（QQ/微信/微博），提示绑定手机怎么办？
    亲爱的齐装网用户，为了您帐号安全及帐号，第一次使用第三方帐号登录后会要求绑定手机号，此时就会将您的第三方帐号与您之前已有帐号绑定，保障帐号一致性。
  • 登录时候忘记密码，怎么办？
    在齐装/装修说APP登录页面，点击【忘记密码】提交找回密码的流程信息，即可快速找回自己的帐号密码。
  • 手机接收不到验证码，怎么办？

    亲爱的齐装网用户，请按照以下方式进行操作：

    1、运营商服务器繁忙，请稍后再次进行验证；

    2、请查看是否被软件拦截；

    3、网络环境问题导致；

    4、您可以发送邮件至sam@qizuang.com说明帐号及手机号码，或齐装网前台“在线客服咨询”寻求帮助；

帐号安全遇到疑惑了？
  • 如何提升帐号安全？
    亲爱的齐装网用户，建议您绑定密保手机、更换安全系数更高的密保邮箱（如腾讯邮箱、新浪邮箱），下载齐装APP，提高帐号的安全系数。
  • 原绑定手机停用，换绑手机怎么办？
    若手机已停用，建议您发送邮件至sam@qizuang.com说明帐号及手机号码申诉，若手机仍可使用，在登录页依次点击【帐号设置】-【密保认证】-【换绑手机号】中修改信绑定手机号。
  • 帐号被盗发布不良信息，如何提高帐号安全度，怎么办？
    亲爱的齐装网用户，建议您不要把帐号信息泄露给他人，并通过以下方式来提高帐号的安全性：绑定手机和邮箱、设置密保认证、下载齐装APP、装修说APP及时发现账号异常。
  • 齐装帐号被盗了，怎么办？

    亲爱的齐装网用户：

    1、若齐装帐号被盗，帐号还可以登录的情况下，请及时修改密码更换密保认证。修改方法：使用齐装app点击【修改密码】

    2、若齐装帐号被盗，帐号无法登录，建议您通过申诉的方式找回帐号。齐装帐号申诉流程：使用齐装app或齐装网网站，在登录页点击【忘记密码 】进行申诉，或使用在线客服协助。

解换绑遇到问题了？
  • 如何对手机/邮箱进行解换绑？
    为了保障用户的帐号安全，密保手机无法解绑，只能换绑。使用齐装app进入【我的】-点击【设置】-进入【绑定手机】进行更换、齐装网用户中心邮箱可以操作解绑后重新绑定。原绑定手机停用，换绑手机怎么办？若手机已停用，建议您发送邮件至sam@qizuang.com说明帐号及手机号码申诉，若手机仍可使用，在登录页依次点击【帐号设置】-【密保认证】-【换绑手机号】中修改信绑定手机号。
  • 齐装帐号解绑规则？
    亲爱的齐装用户，齐装帐号绑定方式包括：手机和邮箱。为了帐号安全，齐装帐号不支持同时解绑所有绑定关系，解绑时必须保留至少一种绑定方式。
  • 绑定手机时，提示验证频繁，无法获取验证码，怎么办？
    亲爱的齐装用户，如您在解换绑操作过程中，系统提示【验证次数过多，请稍候再试】，建议耐心等待24小时后再次尝试，期间不要重复操作，避免延长保护时间。
  • 解换绑的过程中的验证手机/邮箱却不可用，怎么办？
    如您的绑定手机/邮箱不可正常使用， 建议您提交申诉， 申诉成功后， 可根据提示修改。 齐装帐号申诉流程：发送邮件至sam@qizuang.com说明帐号及手机号码申诉，收到反馈邮件后即可登录修改认证。
其他问题解答
  • 如何修改我的昵称？
    亲爱的齐装用户，用户名是唯一找回账号的依据，齐装账号不支持修改，您可以选择打开齐装APP-【我的】-点击【头像】图片-【编辑资料】-【修改昵称】；或登录齐装网平台，进入个人中心基本信息，修改昵称。
  • 搜索到不良信息，怎么办？
    亲爱的齐装用户您好，反馈须知：如果您在使用齐装搜索中，发现不良信息或广告，可以按如下方式发起举报：进入齐装APP-【意见反馈】中提交您的反馈内容，我们会依照相应的法律法规对您投诉的搜索结果链接进行处理，请您知悉。
