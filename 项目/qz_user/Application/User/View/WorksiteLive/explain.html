什么是装修现场？

齐装网服务升级，“装修现场”功能震撼上线。

装修现场是帮助施工人员将工地施工进度、施工现场详情以图文、视频的形式，及时、有效的传达业主和装修公司。

打通装修公司、业主、施工人员之间的信息传递壁垒，帮助业主以及装修公司及时了解装修现场详情，保证装修工程的顺利进行，真正做到互联网装修新未来。

订单数据安全

装修公司通过授权施工人员唯一施工编号的方式，给到施工人员查看编辑施工信息的权限，施工编号具有唯一性，做到可及时追溯，确保订单数据安全，保障装修公司以及业主的切身利益。

如何使用？

1、装修公司确认签约订单（由齐装网分配的订单、主动咨询签单）,齐装官方回访业主后，成功签约的订单将自动作为施工订单。

2、装修公司通过施工编号或施工二维码，授权旗下施工人员。

3、施工人员获得授权码，通过微信“齐装网施工跟进”，查看已签约的施工订单，发布时装修现场信息。
